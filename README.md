# 🧠 Tic Tac Toe Game

A simple implementation of the classic **Tic Tac Toe (Noughts and Crosses)** game where two players take turns marking spaces in a 3×3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

---

## 🎮 Features

- Two-player mode
- Simple and intuitive user interface
- Win/draw detection
- Visual feedback on game result
- Easy to reset and play again

🧠 Game Rules
The game is played on a grid that's 3 squares by 3 squares.

Player 1 is X, Player 2 is O.

Players take turns putting their marks in empty squares.

The first player to get 3 of their marks in a row (vertically, horizontally, or diagonally) wins.

If all 9 squares are filled and neither player has 3 in a row, the game ends in a draw.
